Netflix Scheduled Downloading

becoming more on the go.

Overview

By creating a new creative feature on Netflix, scheduled download allows the user to be more efficient and on the go. Currently, Netflix has smart downloads that automatically download the next episode, or the user will have to download each episode manually. The schedule download feature allows the user more flexibility when episodes will be downloaded and how much watch time they desire.
